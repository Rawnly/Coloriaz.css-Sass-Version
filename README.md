# Colors-Variables
##How to
Copy and paste this: 
```sass
@import 'https://cdn.rawgit.com/Rawnly/Colors-Variables/master/colors.scss';
```
and then you can use it like 

```sass
.foo {
  color: white;
  background-color: $teal-10;
}
```

##Availables Colors
